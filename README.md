 MuSuBi（結び）

音楽で人と人を結ぶ、旅するように聴く音楽コミュニティ

1. プロジェクト概要

**MuSuBi（結び）**は、
再生数やランキング競争ではなく、
「音楽を旅するように楽しむ」ことを目的とした音楽共有Webアプリケーションです。

ユーザーは

好きなYouTube音楽

自作曲・カバー曲

を自由に共有し、
コメントや「いいね（共感）」を通じて音楽でつながることができます。

2. 制作背景（なぜ作ったか）

私は音楽を専攻しており、
現代の音楽プラットフォームにおける

再生数操作

ストリーミング数至上主義

数字だけが評価される環境

に強い違和感を覚えていました。

「音楽は競争ではなく、体験や記憶を共有するものではないか」

という考えから、
ランキングよりも“共感”を軸にした音楽サービスを自分の手で実装してみたいと思い、本プロジェクトを制作しました。

3. サービス名について

**MuSuBi（結び）**という名前は、

Music（音楽）

結び（人と人をつなぐ）

という意味を掛け合わせた造語です。

音楽を通じて人が自然につながる、
そんな空間を表現しています。
